Как сделать заказ?

Оформление заказа осуществляется при помощи Корзины.

Корзина – это удобный инструмент для совершения нескольких покупок одновременно и оформления заказа удаленно.

После того как Вы нашли нужный Вам товар, чтобы добавить его в корзину: нажмите на кнопку с надписью «В корзину» (рядом с товаром) — товар автоматически попадает в корзину.

Вы можете добавить в корзину любой товар из любой категории.

Чтобы посмотреть добавленные в корзину товары нажмите на ссылку «Моя корзина» (под изображением корзины в правой колонке сайта).

Чтобы убрать ненужный товар из корзины — перейдите по ссылке «Моя корзина», и в открывшемся окне, напротив не желаемого товара, кликните по изображению X — удалить товар из корзины.

Когда в корзине будут все нужные товары, можно перейти к оформлению заказа.

Для осуществления заказа откройте корзину («Моя корзина»). В появившемся окне перейдите по ссылке «Оформить заказ». Затем заполните все поля, отмеченные красной звездочкой * (это обязательные поля).

После заполнения всех обязательных полей нажмите кнопку «Подтвердить заказ». После этого по указанному номеру телефона с Вами свяжется наш оператор, а если при оформлении заказа вы указали e-mail, на него придет подробная информация о заказе.
